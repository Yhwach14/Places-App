# Places App

A lightweight Flutter application built to demonstrate seamless integration between cross-platform UI and native device hardware. This project focuses on the intersection of location services and media capture.

## Features

1. Real-time GPS Tracking
2. Native Camera Integration
3. Data Presistence: linking images with location metadata
4. Permission Handling
